# v0.129.0-alpha-termux

This alpha build merges upstream OpenAI Codex `rust-v0.129.0-alpha.6` into the Android Termux fork and packages it as `0.129.0-alpha-termux` for npm `next` validation only.

No GitHub release is planned for this build. Promotion to public `main`/GitHub stays blocked until the npm package has passing Termux test reports.

Install test build:

```bash
npm install -g @mmmbuto/codex-cli-termux@next
```

Termux delta kept:

- Android browser login through `termux-open-url`.
- Fork update channel and npm package name.
- Android package launchers with `CODEX_SELF_EXE`, `LD_LIBRARY_PATH`, bundled `libc++_shared.so`, and `RUNPATH=$ORIGIN`.
- Voice/realtime audio disabled for Android Termux builds.
