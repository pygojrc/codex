# v0.126.0-termux

This release candidate merges current upstream OpenAI Codex `main` into the Android Termux fork and packages it as `0.126.0-termux`.

## Fork delta

- Android/Termux native build target: `aarch64-linux-android`.
- Termux npm package and update flow: `@mmmbuto/codex-cli-termux`.
- Fork-owned rusty_v8 Android artifact resolution.
- Voice/realtime capture disabled on Android builds.

## Install

```sh
npm install -g @mmmbuto/codex-cli-termux@latest
```

For staged testing before promotion:

```sh
npm install -g @mmmbuto/codex-cli-termux@next
```
