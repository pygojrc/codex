# v0.128.0-termux

This release merges upstream OpenAI Codex `rust-v0.128.0` into the Android Termux fork and packages it as `0.128.0-termux`.

It preserves the upstream feature set while keeping the fork-specific Android/Termux packaging and update paths.

## Install

```bash
npm install -g @mmmbuto/codex-cli-termux@latest
```
